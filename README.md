## 🌿 Estructura del Repositorio
El repositorio está organizado por **ramas (branches)** para mantener la independencia de cada entrega:

* `main`: Contiene el código estable y final de la materia.
* `TP-1`: Desarrollo del Trabajo Práctico N° 1.
* `TP-2`: Desarrollo del Trabajo Práctico N° 2.
* `clase-XX`: Ejercicios específicos de clases particulares.

> **Nota:** Para ver un trabajo específico, cambia de rama usando el selector de GitHub arriba a la izquierda.

---

## 🚀 Cómo ejecutar los proyectos
1. Clona el repositorio: 
   `git clone https://github.com/lazartej71/Programacion-3-Teoria-.git`
2. Cambia a la rama deseada:
   `git checkout nombre-de-la-rama`
3. Abre el archivo de solución (`.sln`) en Visual Studio.
